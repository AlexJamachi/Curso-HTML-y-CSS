El repositorio actúa como un directorio central ("Landing Page") que alberga **tres proyectos independientes**, permitiendo visualizar mi evolución en diseño UI/UX y maquetación estructurada.

🔗 **[¡Mira el directorio en vivo aquí!](https://alexjamachi.github.io/Curso-HTML-y-CSS/)**

## 💻 Proyectos Incluidos

### 🐉 1. KaiJu Tech (E-Commerce Landing Page)
Una landing page de alto rendimiento para una tienda ficticia de hardware gaming.
- **Diseño:** Tema oscuro agresivo con acentos en naranja neón y púrpura. Tipografías 'Orbitron' y 'Rajdhani'.
- **Características:** Menú hamburguesa responsivo, efectos *glassmorphism* y maquetación avanzada con CSS Grid y Flexbox.

### 📁 2. Portafolio Web - Versión 2.0 (La mejora)
Una versión completamente rediseñada bajo un formato *Single-Page Application* utilizando HTML5 y CSS3 puros avanzados.
- **Diseño Moderno:** Tema oscuro con acentos púrpuras, enfocado en mi perfil de Backend y Ciberseguridad.
- **UI/UX Mejorado:** Implementación de animaciones al hacer scroll, variables de CSS y diseño 100% responsivo moderno.

### 📁 3. Portafolio Web - Versión 1.0 (El origen)
Es un sitio estático de 3 páginas diseñado para presentar mi perfil profesional.
- **Estructura:** Navegación multipágina (`index.html`, `proyectos.html` y `contacto.html`).
- **Diseño:** Paleta de colores basada en tonos azul técnico y tipografía sin remates.
