¡Hola! 👋 Bienvenido al repositorio de mi portafolio web personal. Este proyecto fue desarrollado como parte de mi formación en la Carrera de Informática de la FCPN (UMSA)[cite: 3].

Para esta entrega, el repositorio ha sido reestructurado para albergar **dos versiones** del portafolio simultáneamente. Esto permite visualizar la evolución, mejora y contraste en el diseño y código.

🔗 **[¡Mira el sitio en vivo aquí!](https://alexjamachi.github.io/Readme2.1/)**

## 🚀 Sobre las Versiones

### 📁 Versión 1.0 (El origen)
Es un sitio estático de 3 páginas diseñado para presentar mi perfil profesional[cite: 3].
- **Estructura:** Navegación multipágina (`index.html`, `proyectos.html` y `contacto.html`)[cite: 3].
- **Diseño:** Paleta de colores basada en tonos azul técnico y tipografía sin remates[cite: 3].
- **Maquetado:** Uso intensivo de etiquetas semánticas HTML5 y diseño mediante anchos fluidos[cite: 3].

### 📁 Versión 2.0 (La mejora)
Una versión completamente rediseñada bajo un formato *Landing Page* unificado, utilizando HTML5 y CSS3 puros avanzados[cite: 3].
- **Diseño Moderno:** Tema oscuro "Dark Mode" con acentos púrpuras, enfocado en mi perfil de Backend y Ciberseguridad.
- **UI/UX Mejorado:** Implementación de animaciones al hacer scroll, efectos de desenfoque (glassmorphism), variables de CSS y un diseño 100% responsivo moderno.
- **Navegación:** Menú fijo superior con *smooth scrolling* hacia las distintas secciones de una sola página.

## 🛠️ Tecnologías y Herramientas
* **Lenguajes:** HTML5 y CSS3 puros (sin frameworks)[cite: 3].
* **Despliegue:** GitHub Pages (con enrutamiento mediante carpetas)[cite: 3].
