## 🚀 Sobre el Proyecto

Este es un sitio estático de 3 páginas diseñado para presentar mi perfil profesional, habilidades tecnológicas y medios de contacto. El objetivo principal es aplicar buenas prácticas de desarrollo web, estructuración semántica y diseño unificado.

### 🌟 Características Destacadas
- **Navegación Fluida:** Menú unificado para transitar entre `index.html`, `proyectos.html` y `contacto.html`.
- **Diseño Responsivo:** Estilos CSS centralizados que adaptan el contenido mediante un diseño de anchos fluidos y elementos flotantes.
- **Estructura Semántica:** Uso intensivo de etiquetas HTML5 (`<header>`, `<main>`, `<section>`, `<figure>`, `<figcaption>`).
- **Formulario Integrado:** Sección de contacto lista para integrarse con un backend en el futuro.

## 🛠️ Tecnologías y Herramientas
* **Lenguajes:** HTML5 y CSS3 puros (sin frameworks).
* **Despliegue:** GitHub Pages.
* **Diseño:** Paleta de colores profesional basada en tonos azul técnico (`#2980b9`, `#2c3e50`) y tipografía sin remates para mejor legibilidad.
